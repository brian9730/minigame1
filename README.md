# 오범투 규칙

## 보드게임 규칙

---

## 게임 소개
**이 게임은 구룡투 보드게임을 벤치마킹하여 제작한 오범투 게임입니다.**

---

## 게임 준비
- **게임을 시작하기 전에 맑은 정신상태를 만들기 위해 세수를 합니다.**
- **맑은 정신상태를 만들었다면 게임시작 버튼을 누르시면 됩니다.**

---

## 게임 플레이
1. **카드 배치버튼을 누르면 플레이어와 상대플레이어에게 각각 1~5가 적힌 카드가 지급됩니다.**
2. **플레이어가 1~5 중에서 원하는 숫자를 제출하면 상대플레이어도 1~5 중 원하는 숫자를 이어서 제출합니다.**
3. **플레이어가 제출한 숫자와 상대플레이어가 낸 숫자를 비교하여 큰 값을 제출한 플레이어가 1점을 얻습니다.**

---

## 게임 종료
- **플레이 중 3점을 먼저 획득하는 플레이어가 승리하게 됩니다.**
- **만약 플레이어와 상대플레이어 점수가 동일할 경우에는 다시시작 버튼을 누르면 됩니다.**
